No Connection To The Internet
Error Details

This Display has a connection to your network but no connection to the internet.

The connection to the outside world is needed for updates and keeping the time.

How To Solve?

Please check your mobile data. you must need to have data connection for accessing the internet.

Go to Settings>More>Mobile Networks and make sure mobile data is turned on. if your mobile data or wifi is turned off, please trun it on and try again to fix this problem.
